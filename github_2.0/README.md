# V-Replenishment Tool

![V-Replenishment Banner](https://img.shields.io/badge/V--Replenishment-Tool-blue)
![Version](https://img.shields.io/badge/version-1.0-green)
![License](https://img.shields.io/badge/license-MIT-lightgrey)

门店智能补货清单生成器 - 自动识别新品和缺货产品，生成专业补货清单

## 🚀 功能特性

### 核心功能
- 📊 **智能数据匹配**：通过SKU/Barcode自动匹配库存和到货数据
- 🆕 **新品智能识别**：自动识别库存中没有的新品并标注"NEW"
- 📝 **专业Excel导出**：等线字体、行颜色交替、新品标黄
- 🎯 **优先级计算**：自动计算补货优先级（新品 > 缺货 > 库存不足）

### 特色功能
- 🖱️ **拖拽上传**：支持Excel/CSV文件拖拽上传
- 🔍 **智能搜索**：实时搜索过滤产品
- 📱 **响应式设计**：适配电脑、平板、手机
- 🖨️ **打印优化**：专业打印格式输出
- 📊 **数据统计**：实时显示统计信息

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **数据处理**: SheetJS (Excel文件处理)
- **部署**: Netlify (静态网站托管)
- **图标**: Font Awesome 6.0
- **字体**: 等线字体 (DengXian)
